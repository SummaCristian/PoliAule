# PoliAule

Find available classrooms at Politecnico di Milano.

> [!WARNING]
> **This is a beta version.** 
> 
> Things can and will change. 
> 
> Not affiliated with Politecnico di Milano.

---

## Usage

1. Select a campus, date, and time range
2. Hit **Search**
3. Browse available classrooms

## Data

Classroom availability is fetched from a live data source and timestamped at generation time. Data may not reflect real-time changes.
